# 记录笔记与 Blog 的仓库

- 使用了 VuePress 生成静态站点
- 源码存放在 master 分支
- 站点部署在 gh-pages 分支

1. `git clone git@github.com:mx623303468/notes-and-blog.git`

2. `cd notes-and-blog`

3. `npm install` 安装依赖

4. `npm run dev` 启动本地开发

5. `npm run build` 打包生成站点

6. `npm run deploy` 提交源码到 master， 并提交打包生成站点， 提交到 gh-pages 分支
